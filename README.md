# Cat Shop

Online store of purebred cats.

---

**Demo**: https://cat-shop.emilhumbatov.com <br>

---

## Installation

 Clone repository
```bash
$ git clone https://github.com/cmdtorch/cat_shop.git
```

 Move to project directory
```bash
$ cd ./cat_shop
```

 Build and run project:
```bash
$ docker compose up --build
```
